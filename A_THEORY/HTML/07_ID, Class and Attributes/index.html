ID - An ID is unique to each element, allowing you to target and style it individually.

Class- A class can be shared by multiple elements, allowing them to be styled or manipulated together using CSS or JavaScript. Multiple classes can be assigned to a single HTML element by separating each class name with a space.

If multiple elements share the same class, their styling properties—such as color, font size, and spacing—can be controlled collectively by targeting that class in CSS. This allows for consistent and efficient styling across all elements with the same class name.
